# P7 – Web Quiz App

A responsive **web-based quiz application** built using **HTML, CSS, and JavaScript**. The app presents multiple-choice questions, provides instant feedback, tracks the user’s score, and allows replay.

## Features
- Multiple-choice quiz questions
- Instant feedback for correct & incorrect answers
- Automatic score calculation
- Final score summary screen
- Restart quiz option

## Tech Stack
- **HTML5**, **CSS3**, **JavaScript (Vanilla)** 

## How It Works
1. A question with four options is displayed
2. User selects an answer
3. Correct/incorrect feedback is shown instantly
4. Score is updated in real time
